## GUI View Guide
This guide provides an overview of the GUI view for the calendar application, explaining its usage
and detailing its functionalities.

### To Create a New Event:
- Click on the `Create Event` button located in the top left corner of the GUI view.
- The user can fill in these properties (`Subject`, `Start`, and `End` fields are required),
  and then select `Create` at the bottom to create the event.
- For optional properties such as `Description`, `Location`, `Status` the user can choose to leave 
them empty, or select `N/A`.

### Change Displayed Events:
- To change the starting date for the list of events to display, click on the 
`Change Starting Date View` button at the bottom, which will display a popup allowing you to change it.

### To Edit an Event:
- Click on the event you want to edit.
- A popup will appear, displaying the event info and letting the user select the property they wish to edit. 
- Then the program will show the current value of the property, allowing the user to change it.

### To Create & Change Calendars:
- The user can create or change the calendar through the dropdown menu in the top right corner.
- To create a new calendar, select the `New Calendar` option (it will always be the last option).
- A popup will appear, prompting the user to enter a name and timezone for the new calendar.
- The user can switch between calendars and their corresponding events will appear from the
  selected start date.
